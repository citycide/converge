# @converge/plugin-five

> High-five plugin for the converge Twitch bot.

## features

* totally imaginary high-fives
* sometimes low-fives
* some surprises

## usage

To give a specified user a high-five:

```
!five [target]
```

For users with the right privileges:

```
!five add (message)
!five remove (id)
!five edit (id) (new message)
```
